# STS2 LAN Connect 使用说明

## 进入大厅

1. 启动游戏并进入多人首页
2. 点击 `游戏大厅`
3. 直接在大厅里完成建房、刷新和加入

## 大厅列表操作

- 大厅左侧使用长条房间卡片列表
- 单击房间卡片会选中目标房间
- 双击房间卡片会直接尝试加入
- 右侧按钮会对当前选中的房间生效

## 房主流程

1. 打开 `游戏大厅`
2. 点击 `创建房间`
3. 填写房间名，可选填写密码
4. 发布成功后，客户端会自动：
   - 启动本地 ENet Host
   - 向大厅服务注册房间
   - 持续发送心跳保活

## 玩家流程

1. 打开 `游戏大厅`
2. 点击 `刷新大厅`
3. 选择目标房间并加入
4. 如果房间有密码，按提示输入密码

## 多人续局

- 房主重新进入一个已存在的多人续局存档时，当前续局会自动重新发布到大厅
- 这个绑定会沿用该续局存档对应的房间信息，而不是要求重新手动建房
- 如果该续局还有多个空闲角色槽位，加入方会先看到角色选择，再进入联机

## 加入等待提示

- 点击加入房间后，会出现加载中的提示弹窗
- 弹窗会按阶段更新，例如：
  - 正在向大厅申请加入
  - 大厅已响应，开始连房主
  - 正在尝试公网 / 局域网候选地址
  - 直连超时后自动尝试备用连接路径
  - 连接成功，进入联机界面

## 网络说明

- 当前大厅模式是 `ENet` 直连优先，失败时会自动切到服务端 relay fallback
- 房主机器如果能让 `33771/UDP` 直连可达，加入速度会更快
- 服务端如果启用了 relay fallback，需要放行 `39000-39063/UDP`
- `WS /control` 主要承担大厅协调与房主会话保活，不替代游戏联机数据通路

## 设置说明

- 普通玩家通常只需要填写 `玩家名`
- 只有排障或切换环境时，才需要展开 `开发网络设置`
- 如果安装包已经附带 `lobby-defaults.json`，默认大厅地址会自动生效，但不会默认明文展示在 UI 中

## 常见问题

### 大厅里看不到房间

- 先确认大厅服务健康检查正常
- 再确认房主房间是否发布成功
- 最后确认房主机器的 `33771/UDP` 是否可达，或服务端的 relay 端口段是否已经开放

### 提示 MOD 不一致

- 所有联机玩家必须使用完全相同版本的 `STS2 LAN Connect`

### 无法重新加入多人续局

- 先让房主重新进入对应的多人续局存档
- 等大厅里重新出现该续局房间后，再由队友加入
- 如果弹出角色选择框，请选择一个未被占用的角色槽位

### 需要回退到手动 LAN/IP

- 官方 Host / Join 页面里的手动 LAN 调试入口仍然保留，可作为排障回退方案
