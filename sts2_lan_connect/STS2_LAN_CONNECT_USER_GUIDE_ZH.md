# STS2 LAN Connect 使用说明

## 进入大厅

1. 启动游戏并进入多人首页
2. 点击 `游戏大厅`
3. 直接在大厅里完成建房、刷新和加入

## 房主流程

1. 打开 `游戏大厅`
2. 点击 `创建房间`
3. 填写房间名，可选填写密码
4. 发布成功后，客户端会自动：
   - 启动本地 ENet Host
   - 向大厅服务注册房间
   - 持续发送心跳保活

## 玩家流程

1. 打开 `游戏大厅`
2. 点击 `刷新大厅`
3. 选择目标房间并加入
4. 如果房间有密码，按提示输入密码

## 网络说明

- 当前大厅模式仍然是 `ENet` 直连优先
- 房主机器仍需保证 `33771/UDP` 可达，或使用虚拟局域网
- `WS /control` 主要承担大厅协调与房主会话保活，不替代游戏联机数据通路

## 设置说明

- 普通玩家通常只需要填写 `玩家名`
- 只有排障或切换环境时，才需要展开 `开发网络设置`
- 如果安装包已经附带 `lobby-defaults.json`，默认大厅地址会自动生效，但不会默认明文展示在 UI 中

## 常见问题

### 大厅里看不到房间

- 先确认大厅服务健康检查正常
- 再确认房主房间是否发布成功
- 最后确认房主机器的 `33771/UDP` 是否可达

### 提示 MOD 不一致

- 所有联机玩家必须使用完全相同版本的 `STS2 LAN Connect`

### 需要回退到手动 LAN/IP

- 官方 Host / Join 页面里的手动 LAN 调试入口仍然保留，可作为排障回退方案
